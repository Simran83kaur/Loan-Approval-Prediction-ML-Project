# Loan-Approval-Prediction-ML-Project
This project focuses on predicting whether a loan should be approved or not using Machine Learning techniques.

## Problem Statement
Financial institutions face challenges in deciding loan approvals. This project helps automate that process using data-driven insights.

## Objective
To build a machine learning model that predicts loan approval based on applicant details.

##  Dataset Features
- Gender
- Married
- Dependents
- Education
- Self_Employed
- ApplicantIncome
- CoapplicantIncome
- LoanAmount
- Loan_Amount_Term
- Credit_History
- Property_Area
- Loan_Status

##  Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn


## Workflow
1. Data Collection
2. Data Understanding (EDA)
3. Data Cleaning
4. Feature Engineering
5. Model Training
6. Model Evaluation

##  Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- Gradient Boosting

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score

## Result
The model successfully predicts loan approval with good accuracy and performance.
